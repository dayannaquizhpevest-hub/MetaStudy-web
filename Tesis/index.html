Guía Digital Didáctica de Blender

Proyecto de tesis con aprendizaje interactivo y visualización en Realidad Virtual

Ver Proyecto

📘 Información del Proyecto

Este proyecto consiste en el desarrollo de una guía digital didáctica para el aprendizaje del software Blender, incluyendo prácticas guiadas y visualización de entornos en realidad virtual (VR).

Objetivo General

Crear una guía educativa que enseñe los fundamentos con su aplicación práctica en Blender 3D, incorporando herramientas de realidad virtual para mejorar el aprendizaje interactivo y las habilidades digitales de cada uno de los estudiantes.

Objetivos Específicos

  • Describir las funciones y herramientas básicas y necesarias para poder elaborar diseños simples para que los estudiantes aprendan el ámbito del diseño digital y la animación.
  • Implementar elementos de realidad virtual para proyectar los modelos 3D a través de lentes VR para un aprendizaje dinámico
  • Integrar los diseños para crear modelos tridimensionales básicos, tener un uso responsable y educativo de las herramientas digitales en el área de Informática.
  • Fomentar el aprendizaje autónomo mediante la exploración de entornos virtuales, potenciando la creatividad y el desarrollo de espacios digitales aplicados al diseño 3D

Prácticas

Modelado de objetos básicos
Escenarios 3D para VR
Animaciones simples
